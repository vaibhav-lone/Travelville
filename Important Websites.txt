threee javascript scripts)
a. respond.js - enables the browser to 	understand and execute CSS media queries which we need for responsive web design.
---> <script src="//cdn.jsdelivr.net/respond/1.4.2/respond.min.js"></script>
b. html5shiv - this enables us to use and style HTML5 elements in older browsers that were available be4 html5 was existed.
---> <script src="//cdn.jsdelivr.net/respond/3.7.2/html5shiv.min.js"></script>
c. selectivizer - this allows us to use css 3 pseudo class like first child and last child 
---> <script src="//cdn.jsdelivr.net/selectivizr/1.0.3b/selectivizr.min.js"></script>
